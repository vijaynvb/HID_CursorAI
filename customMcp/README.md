# Custom MCP

This example demonstrates how to create a custom Model Context Protocol (MCP) server using Node.js and the `@modelcontextprotocol/sdk`. The server exposes several tools and a resource that can be accessed by MCP clients.

This Directory Contains:

- `custom-mcp-node/`: A Node.js project implementing a custom MCP server with Express.
- `custom-mcp-spring-boot-mcp/`: A Spring Boot project implementing a custom MCP server in Java.
